# Project Orchestrator v3

A command-line orchestration system integrating Open WebUI with n8n for 73+ automation commands.

## Features
- 73+ Commands in logical categories
- n8n Integration for workflow automation
- Open WebUI Compatible
- Security-First design

## Installation
1. Set up n8n Workflow with Webhook
2. Add JavaScript code nodes
3. Install Open WebUI Pipe
4. Configure environment variables
